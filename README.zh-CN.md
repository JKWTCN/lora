# LORA Organizes & Runs Apps

> [English](./README.md) | [简体中文](./README.zh-CN.md)

> 一个基于 **Tauri + Vue 3** 的现代化应用启动器，支持分类管理和快速启动，助力高效桌面工作流。

## 🖼️ 项目简介

LORA 让你像管理手机 App 一样，轻松分类、搜索和启动 Windows 程序。界面简洁，操作流畅，支持拖拽添加和批量管理。

## 🚀 功能亮点

- **拖拽添加应用**：支持多种可执行文件，批量拖拽，自动验证类型。
- **分类管理**：自定义分组，右键快速移动，支持重命名和删除。
- **快速启动**：单/双击启动，支持管理员权限，右键更多选项。
- **智能搜索**：实时搜索，快捷键聚焦。
- **现代化界面**：自定义标题栏，响应式设计，流畅动画。
- **轻量级**：基于 Tauri 构建，资源占用少，启动速度快。
- **跨平台**：目前针对 Windows 优化，未来支持跨平台。

## 🛠️ 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **后端**: Rust + Tauri
- **样式**: CSS3 + Flexbox/Grid
- **包管理器**: pnpm

## 📦 安装与使用

### 环境要求

- Node.js >= 18
- Rust >= 1.60
- pnpm >= 8

### 开发环境运行

```bash
# 克隆仓库
git clone https://github.com/JKWTCN/lora.git
cd lora

# 安装依赖
pnpm install

# 开发模式运行
pnpm tauri dev
```

### 构建发布版本

```bash
# 构建生产版本
pnpm tauri build

# 构建后的应用程序将在 src-tauri/target/release/bundle/ 目录中
```

## 🎯 使用方法

### 入门指南

1. **启动 LORA**：安装后运行应用程序
2. **添加应用**：将可执行文件（.exe、.bat 等）拖拽到主窗口，或右键创建新项目
3. **组织管理**：创建自定义分类来组织您的应用程序
4. **启动应用**：单击或双击启动应用程序

### 高级功能

- **批量操作**：选择多个应用进行移动、删除或修改属性
- **管理员模式**：右键选择"以管理员身份运行"获取提升权限
- **搜索功能**：使用 Ctrl+F 快速查找应用程序
- **自定义图标**：应用程序自动提取图标用于显示

## 🔧 快捷键

- `Esc`：隐藏搜索框
- `Ctrl+F`：打开搜索
- `Enter`：启动选中的应用程序
- `Delete`：删除选中的应用程序（需确认）

## ❓ 常见问题

> Q: 启动报错或无法添加应用？
> A: 请确认已安装最新 Node.js、Rust，并以管理员权限运行。

> Q: 如何批量添加应用？
> A: 支持直接拖拽多个文件到主窗口。

> Q: 默认分组无法删除？
> A: 为防止误操作，默认分组不支持删除。

> Q: 可以自定义外观吗？
> A: 目前 LORA 使用固定的现代主题。自定义选项可能会在未来版本中添加。

> Q: 数据是否可移植？
> A: 是的，您的应用数据存储在本地，可以通过复制应用程序数据目录进行备份。
